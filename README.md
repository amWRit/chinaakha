# Chinaakha - Nepali Poetry Website

## Features

- Pixel-perfect public UI for Nepali poems and images
- Progressive loading and responsive design
- Admin dashboard for managing poems and admins
- Add, edit, and delete text/image poems
- Add and manage admin users
- Google Drive integration for image poems
- Authentication for admin access
- Instagram and branding assets integrated

For setup and deployment, see internal documentation or ask the developer.
